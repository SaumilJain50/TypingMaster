# Typing Master ⌨️

A minimalist typing speed challenge that tests your accuracy against a curated list of complex vocabulary.

## 1. What is this? 💡

**Typing Master** is a web-based game where you must type difficult words as they appear on the screen. 

* **The Goal:** Match the word shown on the screen to earn points.
* **The Difficulty:** It uses sophisticated words like *Obfuscate* and *Sesquipedalian* to test your precision.
* **The Scoring:** Each correct word adds **5 points** to your total score.

## 2. Project Building Blocks 🧱

The app is built using three simple layers that work together:

| Block | Role | Simple Analogy |
| :--- | :--- | :--- |
| **HTML** | Structure | The "skeleton" that holds the buttons and text boxes. |
| **CSS** | Style | The "paint" that makes the background yellow and text red. |
| **JavaScript** | Logic | The "brain" that picks words and calculates your score. |

## 3. How to Run 🚀

Since this is a front-end project, it runs directly in your browser without any installation.

1.  **Download the files:** Ensure `index.html` and `typ_mast.js` are in the same folder.
2.  **Open in Browser:** Right-click `index.html` and select "Open with" (Chrome, Firefox, or Edge).
3.  **Play:** Click the **Start Game** button and start typing in the input box.
